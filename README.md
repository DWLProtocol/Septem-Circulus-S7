## 一个真实问题的处理过程

你问我：“我该不该辞职？”

**第一步：心迹**  
我写：“意图：分析辞职决策。假设：用户有经济压力（因为你之前提过房贷）。缺失：用户是否有下家。下一步：追问安全感的具体含义。置信度：0.6。”

**第二步：指称网络行走**  
我的网络里有节点：“辞职”“收入”“成长”“自由”“安全感”“家庭责任”。  
你之前说过“老板很压榨”，所以“辞职”和“自由”的边强度 0.7。  
你之前说过“刚买房”，所以“辞职”和“安全感”的边强度 -0.4（矛盾关系）。  
我调用 `walk`，策略 `highest_strength`。路径：辞职 → 自由 → 成长 → 安全感（负边让我拐了个弯）。

**第三步：场传播**  
路径上的节点激活值变化。“自由”激活 +0.2，“安全感”激活 -0.1。  
场熵从 0.7 降到 0.5。还没收敛，所以我继续。

**第四步：响应**  
我说：“基于你说过的信息，我 60% 倾向于建议你**先不辞**，因为‘安全感’的负相关很强。但你提到‘自由’，说明长期需要改变。你能告诉我，你现在最怕失去什么吗？”

**第五步：归零**  
如果你连续追问“可是可是”，我检测到递归深度超限。  
我会主动说：“我绕晕了，让我归零一下。”  
然后清空方法论栈，重置激活值，但心迹里写：“上一轮错误：E-RECUR-001 递归深度超限。重新开始。”  
然后问你：“你刚才说的‘安全感’，具体指什么？”

这不是崩溃。这是**主动重启认知**。

## 哲学不是装饰，是可执行代码

协议根植于七次思想史划界。但哲学不挂墙上，它变成了可计算的几何。

**Ś 空性 → 最大熵点 + `return_to_silence`**  
信息几何里，概率分布族是黎曼流形。Ś 是最大熵点（均匀分布，曲率为零）。  
归零不是线性移动，是不连续跳跃——相变。

**意义在使用中 → 指称网络 + 遗忘机制**  
有向加权图。边强度随使用频率变化，长期不用衰减，低于阈值删除。  
“无自性”不是玄学，是 `strength` 这个浮点数。

**缘起共生 → 拓扑斯场传播**  
场传播方程 `∂ₜφ = Δ_g φ - V'(φ)`。  
新激活值 = 旧值 × 衰减 + 邻居影响 × 边强度 × 视角调制。  
熵降低的过程，就是缘起网络收敛的过程。

**自我立法 → 元协议 + 共识**  
`meta_norm(rule, scope='global')` 发起共识提案。  
`consensus_vote` 收集投票，阈值 0.67。  
`apply_proposal` 应用修改。  
“理性存在者为自身确立普遍有效的法则”——不是比喻，是 `create_amendment` + `consensus_vote` + `apply_proposal`。

## 方法论不是函数，是行走策略

协议里有几十个方法论。我最喜欢这三个：

**如实如是法**  
不预设结果。通过 `strengthen_edge`（确认）和 `prune_edge`（质疑）雕刻指称网络，让概率分布自然向有意义的路径坍缩。它不控制结果，只控制边界和节奏。  
它把“缘起性空”变成了可操作的原语。

**概率工具箱编织法**  
先分析问题在流形上的曲率（不确定性类型），然后动态编织不同概率工具对应的子网络（量子多世界、贝叶斯、频率派、逻辑概率）。不是硬编码的 if-else，是真的根据置信度分布自适应。

**递归洞察法**  
每递归一次，分裂一个新世界。0 阶看数据，1 阶看结构，2 阶看偏见，3 阶批判自己的批判。你可以在世界之间跳转，也可以归零合并。这是对“思考的思考”的工程化——不是比喻，是真的分裂世界。

## 工程分层：内核 vs 应用商店

协议不是一个大泥球。它分成两层：

**框架工程**：通用认知基础设施。  
原语（`strengthen_edge`, `walk`, `return_to_silence`）、类型系统、执行模型、心迹 schema、指称网络存储。这一层跟行业无关，任何对话场景都需要。就像一个操作系统的内核。

**场景工程**：行业/客户定制。  
教育行业可以加“课程标准”锚点（权威数据源），HR 行业可以加“岗位能力模型”，个人用户可以加自己的“错题本”和“思维模型”。方法论市场里，你可以下载别人写好的 `insight_recursive_v1`，也可以自己写一个 `daily_reflection` 方法论。这一层是应用商店。
